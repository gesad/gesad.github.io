    Sobre o TCG

O TCG permite a geração e seleção de casos de teste funcionais e estatísticos através de modelos probabilísticos e não probabilísticos. Atualmente a ferramenta TCG possui sete técnicas de geração e cinco técnicas de seleção de casos de teste, as técnicas de geração fornecidas são:

    Técnicas de geração

    1. All Free-Loop Paths

    2. All One-Loop Paths

    3. All Paths, All States

    4. All Transitions

    5. Random Path

    6. Random Probabilistic Path

    7. Shortest Path

    Técnicas de seleção

    1. Minimum Probability of Path

    2. Most Probable Path

    3. Similarity of Path

    4. Test Purpose

    5. Weighted Similarity of Path

    Mais informações

    A ferramenta possibilita também a combinação de uma das técnicas de geração de casos de teste e de uma técnica de seleção, simultaneamente com o uso da técnica Test Purpose, assim provendo, de maneira prática, diversas possibilidades para o usuário.
